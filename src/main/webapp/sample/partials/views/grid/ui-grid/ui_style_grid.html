Grid Style 적용
Default style : $scope.$emit('updateCSS', 'css/ui-grid/ui-grid-default.css');
Autumn style : $scope.$emit('updateCSS', 'css/ui-grid/ui-grid-autumn.css');
Sky style : $scope.$emit('updateCSS', 'css/ui-grid/ui-grid-sky.css');

기타 원하는 스타일은 홈페이지를 통해  만들수 있다. (http://ui-grid.info/customizer)
